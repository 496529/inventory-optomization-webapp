# inventory-optomization-webapp
Inventory Management System (Evolution: Desktop to Web)

Original Build: Developed a local desktop application using Python, Tkinter, and SQLite to manage CRUD operations for products, suppliers, and billing.

Current Build (Live Web App): Re-engineered the system into a responsive web application using React, TypeScript, and Tailwind CSS. Migrated from a local SQLite setup to a scalable architecture that supports live deployments, real-time state management with React Query, and a modern, accessible UI.

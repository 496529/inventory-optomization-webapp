# inventory-optomization-webapp
This project began as a desktop-based Python application. While functional, I realized that modern business needs require accessibility from any device and real-time data availability.
Version 1.0 (Legacy): Built with Python (Tkinter) and SQLite Desktop Application. This phase focused on mastering CRUD operations and the core logic of inventory math and role-based security.

Version 2.0 (Current): Re-engineered into a modern web app. I migrated the logic to React and TypeScript, implemented a responsive UI with Tailwind CSS, and deployed the application live.
